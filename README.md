# Proyecto_Sprint_7

Repositorio creado para el proyecto del Sprint 7

Proyecto Sprint 7 – Dashboard de anuncios de coches (Streamlit)

Esta aplicación web (Streamlit) sirve para explorar un conjunto de datos de anuncios de venta de coches.
La app, construye visualizaciones interactivas con Plotly a partir del dataset.

## Funciones

Botón para construir un histograma de `odometer` (kilometraje).
Botón para construir un gráfico de dispersión `odometer` vs `price` (kilometraje vs precio).
